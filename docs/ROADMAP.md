# Roadmap

## 1. Stable Online App

Make LiDAR Studio feel like a complete creative tool on first load:

- demo mode by default
- clear Record3D connection flow
- stable controls
- production deployment path

## 2. Presets

Add a preset system for saving and sharing visual states:

- built-in presets
- local custom presets
- import/export JSON

## 3. Agent Trails

Develop the current trail layer into a fuller temporal system:

- better default flow fields
- independent color controls
- velocity, curl, and attraction modes
- shader or ribbon rendering

## 4. Mesh Surfaces

Add depth-derived surface rendering:

- grid mesh from live/demo depth data
- gap-aware triangle generation
- wireframe and translucent materials
- hybrid mesh + points + trails modes

## 5. Audio Reactivity

Use Web Audio to drive:

- point size
- trail birth rate
- color intensity
- mesh displacement
- fog and post-processing

## 6. Recording And Replay

Support repeatable visual work:

- load recorded frame sequences
- export preset snapshots
- document OBS capture workflow

## 7. Shader And Post Effects

Add a cinematic layer after CPU-side rendering is stable:

- bloom
- vignette
- color grading
- shader displacement
- hallucination/noise modes
